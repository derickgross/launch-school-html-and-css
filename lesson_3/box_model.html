Inline Elements

Inline elements occupy only the space of the content. They do not take width and height values.

Anchor tags are normally inline elements; they only take up the width of their content and not the whole width of the page. The next anchor tags follow right after it.

What happens if we set a margin property on an inline level element?

Inline elements will only implement left and right margin values.

HomePortfolioContact

In the example I put a margin of 30px on all sides. As you can see if you check the anchor tag in the developer tools, only the left and right side of the contents are affected by the margin.
