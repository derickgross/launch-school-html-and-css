Welcome to Shutter Bug, the blogging application that hasn't actually been built yet. Above you see a beautiful picture of some wetlands, which the HTML for this page would have you believe was taken in Long Island, NY, when, in fact, the web developer who built this page (me) has no idea where the image came from.

The point of this page is not to actually build anything of substance, but to practice taking a Photoshop file and converting it into the HTML and CSS elements that will reflect the design identically. If I only had the file to reference I would have to make some assumptions about the behavior of the page, but fortunately Launch School provided a working demo page.
